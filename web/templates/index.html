autodeb

Hello, World!

This server has a dput-compatible API. Try adding this to ~/.dput.cf: 

[autodeb]
fqdn = HOST:PORT
incoming = /test-and-upload
method = http
allow_unsigned_uploads = 0
progress_indicator = 2
# Allow uploads for UNRELEASED packages
allowed_distributions = .

Then, upload a package: "dput autodeb package.changes"

Now, take a look at the logs on the server. You will see PUT requests going trough.

Finally, confirm that the files were received by looking at the "data" directory.

This is it for now, but there is more to come.
