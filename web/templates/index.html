Hello, World!

This server has a dput-compatible API. Try adding this to ~/.dput.cf: 

[autodeb]
fqdn = {{ .ServerURL }}
incoming = /upload
method = http
allow_unsigned_uploads = 1
allowed_distributions = .

Then, upload a package: "dput autodeb package.changes"

If you have access to the server. Confirm that the files were received by looking at the "data" directory.

Now, head over to /uploads and confirm that your upload is listed.

This is it for now, but there is more to come.
